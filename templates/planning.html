Fase de Planificación del Proyecto

La fase de planificación de este proyecto se centró en definir el alcance, los objetivos y las metodologías para analizar la economía virtual de GTA Online. Se investigaron fuentes de datos relevantes, se delinearon los modelos de datos para transacciones, precios de mercado y jugadores, y se estableció la estructura general de la aplicación web.

Objetivos clave de esta fase:

  • Definir el problema a abordar: la falta de herramientas visuales accesibles para entender el impacto de las microtransacciones.
  • Establecer los objetivos generales y específicos del estudio.
  • Identificar las principales entidades del sistema (Jugadores, Items, Transacciones, Precios).
  • Diseñar la arquitectura básica de la aplicación (FastAPI como backend, Jinja2 para el frontend).
  • Planificar la recolección y simulación de datos.

Esta fase fue crucial para sentar las bases sólidas sobre las cuales se construiría el resto del proyecto, asegurando que todos los componentes estuvieran alineados con la visión central.
